**🧮 Programa de cálculo – Consumo de energia**💡

👩🏽‍💻Este programa em Python foi criado para calcular quanta energia um aparelho eletrônico consome, de acordo com o período utilizado. 

➡️Fórmula utilizada: consumoMensal = (potencia * horasDia * 30) / 1000
